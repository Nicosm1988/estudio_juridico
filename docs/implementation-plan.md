# Plan de implementación

1. Base Next.js 16, diseño responsive y contenido provisional centralizado.
2. Chat web determinístico con consentimiento, admisión guiada y persistencia.
3. Panel de recepción para consultas, respuestas, estados y prioridades.
4. PostgreSQL con Drizzle, migraciones, auditoría y retención.
5. Pruebas, preview de Vercel y promoción a producción.

La primera versión excluye expresamente WhatsApp y asesoramiento jurídico por IA.
