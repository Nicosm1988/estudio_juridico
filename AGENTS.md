# Reglas del repositorio

- Mantener TypeScript estricto, App Router y componentes de servidor por defecto.
- No incorporar WhatsApp, Meta API ni asesoramiento jurídico automatizado.
- Toda admisión debe guardar mensajes, consentimiento y estado en PostgreSQL.
- No inventar profesionales, matrículas, domicilios ni promesas comerciales.
- Ejecutar `pnpm lint`, `pnpm test` y `pnpm build` antes de publicar.
